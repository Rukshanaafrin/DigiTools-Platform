### DigiTools Platform

Name of Project :
DigiTools Platform


A Little Description :
A modern and responsive React single-page application for showcasing premium digital tools. Users can browse products, add items to cart, remove selected products, and complete checkout with toast notifications.


Technology That I Used :
- React.js
- Tailwind CSS
- JavaScript (ES6)
- Vite
- React Toastify
- JSON Data


3 Features About My Project :
- Product cart count updates instantly in the navbar
- Users can remove selected products from cart and total updates dynamically
- Proceed to checkout clears all cart items and shows toast alerts
